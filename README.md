# dotfiles
My dotfiles for my Arch Linux Install

## Requirements
**Required:**
1. awesome-git
2. picom-tryone-git
3. pulseaudio or pipewire-pulse
**Optional:**
1. Alacritty

## Using
If you wish to use all the provided dot files, simply run the load.sh script. You may need to add execute permissions to the file using `chmod +x load.sh`

## Contributing
Simply make your changes to your local dot files and run save.sh to copy all the files to the git repo. Make sure your current working directory is the root of the git repo. You may need to add execute permissions to the file using `chmod +x save.sh`
